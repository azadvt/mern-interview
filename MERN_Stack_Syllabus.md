# MERN Stack Complete Course Syllabus

**Duration:** 2 Months | 5 Days/Week | 2 Hours/Day
**Total Classes:** ~44 Days (~88 Hours)
**Instructor:** Azad VT

---

## Week 1: HTML & Web Basics (5 Days / 10 Hours)

| Day | Topic | Hours |
|-----|-------|-------|
| Day 1 | How the Web Works — Browser, Server, HTTP/HTTPS, DNS, Domain, Hosting | 2 |
| Day 2 | HTML Basics — Structure, Tags, Elements, Attributes, Text Tags, Lists | 2 |
| Day 3 | Links, Images, Audio, Video, Tables, Iframes | 2 |
| Day 4 | Forms — Input Types, Labels, Buttons, Select, Textarea, Validation | 2 |
| Day 5 | Semantic HTML, Meta Tags, SEO Basics, Dev Tools, VS Code, Git & GitHub | 2 |

---

## Week 2-3: CSS (8 Days / 16 Hours)

### Week 2: CSS Fundamentals
| Day | Topic | Hours |
|-----|-------|-------|
| Day 6 | CSS Introduction — Selectors, Box Model, Colors, Backgrounds, Borders | 2 |
| Day 7 | Typography, Units (px, em, rem, %, vw, vh), Display, Positioning | 2 |
| Day 8 | Flexbox — Container & Item Properties, Alignment, Ordering | 2 |
| Day 9 | CSS Grid — Template Rows/Columns, Gap, Areas, Auto-fit/Auto-fill | 2 |
| Day 10 | Responsive Design — Media Queries, Mobile-First, Breakpoints | 2 |

### Week 3 (First 3 Days): CSS Advanced
| Day | Topic | Hours |
|-----|-------|-------|
| Day 11 | Transitions, Transforms, Animations, CSS Variables, Pseudo-classes | 2 |
| Day 12 | Bootstrap / Tailwind CSS Basics (Overview) | 2 |
| Day 13 | **Project:** Responsive Landing Page + HTML/CSS Revision | 2 |

---

## Week 3-5: JavaScript (12 Days / 24 Hours)

### Week 3 (Remaining 2 Days) + Week 4: JS Basics & Intermediate
| Day | Topic | Hours |
|-----|-------|-------|
| Day 14 | JS Introduction — Variables (var, let, const), Data Types, Operators | 2 |
| Day 15 | Conditionals (if/else, switch), Loops (for, while, for...of, for...in) | 2 |
| Day 16 | Functions — Declaration, Expression, Arrow Functions, Scope, Hoisting | 2 |
| Day 17 | Arrays — Methods (push, pop, map, filter, reduce, find, forEach, splice, slice) | 2 |
| Day 18 | Objects — Properties, Methods, Destructuring, Spread/Rest Operators | 2 |
| Day 19 | Strings, Template Literals, Type Conversion, Math Object, Date | 2 |
| Day 20 | Closures, Callbacks, Higher-Order Functions | 2 |

### Week 5: JS Advanced
| Day | Topic | Hours |
|-----|-------|-------|
| Day 21 | DOM Manipulation — Selecting, Creating, Modifying, Removing Elements | 2 |
| Day 22 | Events — Listeners, Bubbling, Delegation, preventDefault | 2 |
| Day 23 | ES6+ — Classes, Modules (import/export), Symbols, Iterators | 2 |
| Day 24 | Asynchronous JS — Promises, async/await, Fetch API, Error Handling | 2 |
| Day 25 | LocalStorage, JSON, **Mini Project:** To-Do App / Quiz App | 2 |

---

## Week 6-7: Node.js, Express.js & MongoDB (10 Days / 20 Hours)

### Week 6: Node.js & Express.js
| Day | Topic | Hours |
|-----|-------|-------|
| Day 26 | Node.js — Introduction, NPM, Modules (fs, path, os), package.json | 2 |
| Day 27 | Node.js — Events, Streams, HTTP Module, Creating a Basic Server | 2 |
| Day 28 | Express.js — Setup, Routing, Request/Response, Query & Route Params | 2 |
| Day 29 | Express.js — Middleware, Static Files, Error Handling, MVC Pattern | 2 |
| Day 30 | REST API Concepts — GET, POST, PUT, DELETE, Status Codes, Postman | 2 |

### Week 7: MongoDB & Authentication
| Day | Topic | Hours |
|-----|-------|-------|
| Day 31 | MongoDB — Introduction, CRUD Operations, MongoDB Atlas/Compass Setup | 2 |
| Day 32 | Mongoose — Schema, Model, Validation, Data Types, Default Values | 2 |
| Day 33 | Mongoose — Relationships (ref & populate), Aggregation, Indexing | 2 |
| Day 34 | Authentication — JWT, Bcrypt, Login/Register API, Protected Routes | 2 |
| Day 35 | **Project:** Build Complete REST API (User Auth + CRUD + File Upload) | 2 |

---

## Week 8-9: React.js (12 Days / 24 Hours)

### Week 8: React Fundamentals
| Day | Topic | Hours |
|-----|-------|-------|
| Day 36 | React Introduction — Vite Setup, JSX, Components, Props, Fragments | 2 |
| Day 37 | State — useState, Event Handling, Conditional Rendering, Lists & Keys | 2 |
| Day 38 | useEffect — Side Effects, API Calls (Axios/Fetch), Loading & Error States | 2 |
| Day 39 | React Router — Routes, Links, useNavigate, useParams, Nested Routes | 2 |
| Day 40 | Forms in React — Controlled Components, Form Validation, useRef | 2 |

### Week 9: React Advanced + Full Stack
| Day | Topic | Hours |
|-----|-------|-------|
| Day 41 | useContext — Global State, Theme/Auth Context, Context + useReducer | 2 |
| Day 42 | Custom Hooks, Performance (useMemo, useCallback, React.memo) | 2 |
| Day 43 | Connecting React with Backend — Full Stack CRUD, CORS, Environment Variables | 2 |
| Day 44 | Auth in React — Login/Register UI, Token Storage, Protected Routes | 2 |
